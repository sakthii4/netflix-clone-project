# 🎬 Netflix Clone Project

A Netflix-like website built with **HTML and CSS**.

## 📁 What's Inside

- **index.html** - Main landing/home page
- **signin.html** - Sign in page
- **signup.html** - Sign up page  
- **browse.html** - Browse movies/shows page
- **movie.html** - Individual movie details page
- **style.css** - Styling for the home page
- **browse.css** - Styling for browse page
- **movie.css** - Styling for movie page
- **assets/** - Folder with images and resources

## 🚀 Live Demo

Check it out here: https://netflix-clone-project-ivory.vercel.app

## 💻 Technologies Used

- HTML (68.1%)
- CSS (31.9%)

## 📝 How to Use

1. Clone this repository
2. Open `index.html` in your browser
3. Navigate through the pages - Sign up, Browse movies, and View details
